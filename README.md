# AI Therapist Agent

An AI-powered mental health support web application designed to provide conversational emotional support, stress-related prompts, and wellness assistance through an interactive user interface.

## Features

* AI-based therapeutic conversation interface
* Stress and anxiety keyword detection
* Mental wellness support prompts
* Interactive frontend interface
* Backend API support
* Modular frontend and backend project structure

## Technologies Used

### Frontend

* Next.js
* React
* TypeScript
* Tailwind CSS

### Backend

* Node.js
* TypeScript

## Setup Procedure

### Frontend

```bash
cd ai-therapist-agent-main
npm install
npm run dev
```

### Backend

```bash
cd ai-therapist-agent-backend-main
npm install
npm run dev
```
